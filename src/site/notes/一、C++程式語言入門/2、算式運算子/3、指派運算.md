### 指派運算子 

- **A += B →A=A+B**
- 指派 → A = B 
- 相加並指派 → A += B 
- 相減並指派 → A -= B 
- 相乘並指派 → A * = B 
- 相除並指派 → A / = B 
- 取餘數並指派 → A %= B 
- 或(OR)並指派 → A |= B 
- 且(AND)並指派 → A &= B 
- XOR並指派 → A ^= B 
- 向左位移並指派 → A <<= B 
- 向右位移並指派 → A >>= B
