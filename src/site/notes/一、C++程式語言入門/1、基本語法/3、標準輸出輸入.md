### 輸出

- cout-輸出
- endl-換行

```
cout << "Hello" << endl << "World!";

輸出 →
Hello
World!
```

```
輸出變數 →
int main() {
    int v1 = 100;
    int v2 = 9487;
    cout << v1 << endl << v2;
    return 0;
}

輸出 →
100
9487
```

### 輸入

- cin-輸入

```
int i;
cin >> i;
```
