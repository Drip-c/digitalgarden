### 語法Syntax

- **每行程式後面都需要加上「;」分號，除了#include、#define、{}後面**

### 標頭檔 Headers

- →#include<標頭檔>

```
#include <iostream>
...
#include <cmath>
```

### C++命名空間

- 每個程式皆須加上

```
using namespace std;
```

### 主程式

- Main function-程式起點
- ( )；參數定義
- { }；程式內容

```
int main() {
	...
}
```

### 註解

1. / * 註解 * /
2. // 註解 (僅限該行)
